# Sistema de Gestão de Chamados - Suporte TI

[![MongoDB](https://img.shields.io/badge/MongoDB-%234ea94b.svg?style=for-the-badge&logo=mongodb&logoColor=white)](https://www.mongodb.com/)

Resolução de uma atividade prática de banco de dados NoSQL utilizando o **MongoDB**. O projeto simula o ecossistema de um *Help Desk* de suporte técnico de TI, implementando todas as operações fundamentais do ciclo **CRUD** (Create, Read, Update, Delete) diretamente através do terminal do `mongosh`.

## Como Usar

Para replicar os testes e interagir com o banco de dados, siga os passos abaixo:

1. Certifique-se de ter o **MongoDB** instalado e o serviço rodando localmente (ou utilize um cluster no MongoDB Atlas).
2. Abra o seu terminal e conecte-se ao banco utilizando o **MongoDB Shell**:
```bash
   mongosh
